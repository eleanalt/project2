Περιγραφή του προγράμματος:
  Το πρόγραμμα μας διαβάζει αρχικά δεδομένα από ένα input αρχειο json , που περιέχει όσα αναγράφονται στην εκφώνηση.
Στη συνέχεια εφαρμόζεται τριγωνοποίηση με προσθηκη steiner σημειων για καλύτερη ποιότητα τριγωνοποίησης.
Οι μέθοδοι βελτιστοποίησης που χρησιμοποιούνται είναι:
Τοπική Αναζήτηση (Local Search): Βελτιστοποίηση μέσω της εισαγωγής νέων σημείων και υπολογισμού του κόστους της τριγωνοποίησης.
Προσομοιωμένη Ανόπτηση (Simulated Annealing): Μέθοδος βελτιστοποίησης μέσω θερμοκρασίας και ψύξης για την εύρεση καλύτερης τριγωνοποίησης.
Αποικία Μυρμηγκιών (Ant Colony Optimization): Χρησιμοποιεί φερομόνες για την καθοδήγηση των μυρμηγκιών στη διαδικασία βελτιστοποίησης.
Όλα αυτά υλοποιούνται στο αρχείο TriangulationOptimizer. Παράλληλα στο InputParser γινεται διαχείριση της εισόδου. Στο Output  διαχείριση της εξόδου. και στο main λαμβάνεται η είσοδος χρησιμοποιούμε το triangulationoptimizer και παράγουμε και το αρχείο εξόδου.
